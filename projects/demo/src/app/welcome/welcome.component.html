Welcome

This demonstration application is dedicated to showcasing ngx-matomo features in an almost real application.

Code examples

Setting up Matomo integration 
{{
"matomoConfig = {
  scriptUrl: '//cdn.matomo.cloud/ngx.matomo.cloud/matomo.js',
  trackers: [
    {
      trackerUrl: 'https://ngx.matomo.cloud/matomo.php',
      siteId: 1,
    },
  ],
  skipTrackingInitialPageView: false,
  requireConsent: true,
  routeTracking: {
    enable: true,
  },
  trackLinks: true,
}"
    }}
